> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development to implement this plan task-by-task.

**Goal:** (A) Close the feedback loop so the LLM sees action outputs and users see results. (B) Add browser tool via Playwright. (C) Daemon mode so kernel runs in background.

```rust
pub fn get_task_actions(&self, task_id: &str) -> rusqlite::Result<Vec<ActionRecord>> {
    let mut stmt = self.conn.prepare(
        "SELECT id, task_id, step_index, tool, params_json, result_json, success, duration_ms, created_at
         FROM actions WHERE task_id = ?1 ORDER BY step_index ASC",
    )?;
    let rows = stmt.query_map(params![task_id], |row| {
        Ok(ActionRecord {
            id: row.get(0)?,
            task_id: row.get(1)?,
            step_index: row.get(2)?,
            tool: row.get(3)?,
            params_json: row.get(4)?,

}
```

## Phase B — Browser Tool (Tasks B1-B3)

### Task B1: Playwright Python subprocess bridge

**Files:** `workers/internal/browser/worker.go`

The Go worker will spawn a Python subprocess that uses Playwright. The bridge protocol: Go writes JSON to stdin, Python reads it, executes, writes JSON result to stdout.

- [ ] **Step 1: Create browser bridge Python script**

Create `workers/browser_bridge.py`:
```python
#!/usr/bin/env python3
"""Playwright bridge — reads JSON actions from stdin, writes results to stdout."""
import sys, json

def run_action(action: dict) -> dict:
    from playwright.sync_api import sync_playwright
    
    op = action.get("operation")
    url = action.get("url", "")
    
    with sync_playwright() as p:
        browser = p.chromium.launch(headless=action.get("headless", True))
        page = browser.new_page()
        
        try:
            if op == "navigate_and_extract":
                page.goto(url, timeout=30000)
                text = page.inner_text("body")
                return {"success": True, "output": text[:5000]}
            
            elif op == "screenshot":
                page.goto(url, timeout=30000)
                path = action.get("path", "screenshot.png")
                page.screenshot(path=path)
                return {"success": True, "output": f"screenshot saved to {path}"}
            
            elif op == "click":
                page.goto(url, timeout=30000)
                page.click(action.get("selector", ""))
                text = page.inner_text("body")
                return {"success": True, "output": text[:2000]}
            
            elif op == "fill_form":
                page.goto(url, timeout=30000)
                for selector, value in action.get("fields", {}).items():
                    page.fill(selector, value)
                if action.get("submit"):
                    page.click(action.get("submit"))
                return {"success": True, "output": "form submitted"}
            
            else:
                return {"success": False, "output": f"unknown operation: {op}"}
        except Exception as e:
            return {"success": False, "output": str(e)}
        finally:
            browser.close()

if __name__ == "__main__":
    for line in sys.stdin:
        line = line.strip()
        if not line:
            continue
        try:
            action = json.loads(line)
            result = run_action(action)
        except Exception as e:
            result = {"success": False, "output": str(e)}
        print(json.dumps(result), flush=True)
```

- [ ] **Step 2: Create browser worker Go file**

Create `workers/internal/browser/worker.go`:
```go
package browser

import (
    "bufio"
    "encoding/json"
    "fmt"
    "os/exec"
    "path/filepath"
    "runtime"
)

type Result struct {
    Success bool
    Output  string
}

func Execute(operation string, params map[string]interface{}) Result {
    // Build path to browser_bridge.py relative to the worker binary
    bridgePath := filepath.Join("workers", "browser_bridge.py")
    
    pythonCmd := "python"
    if runtime.GOOS == "windows" {
        pythonCmd = "python"
    }
    
    cmd := exec.Command(pythonCmd, bridgePath)
    
    stdin, err := cmd.StdinPipe()
    if err != nil {
        return Result{false, fmt.Sprintf("failed to get stdin: %v", err)}
    }
    
    stdout, err := cmd.StdoutPipe()
    if err != nil {
        return Result{false, fmt.Sprintf("failed to get stdout: %v", err)}
    }
    
    if err := cmd.Start(); err != nil {
        return Result{false, fmt.Sprintf("failed to start browser bridge: %v", err)}
    }
    
    // Write action to bridge
    action := map[string]interface{}{
        "operation": operation,
    }
    for k, v := range params {
        action[k] = v
    }
    
    data, _ := json.Marshal(action)
    fmt.Fprintf(stdin, "%s\n", data)
    stdin.Close()
    
    // Read result
    scanner := bufio.NewScanner(stdout)
    var resultLine string
    for scanner.Scan() {
        resultLine = scanner.Text()
    }
    cmd.Wait()
    
    if resultLine == "" {
        return Result{false, "no output from browser bridge"}
    }
    
    var result map[string]interface{}
    if err := json.Unmarshal([]byte(resultLine), &result); err != nil {
        return Result{false, fmt.Sprintf("failed to parse result: %v", err)}
    }
    
    success, _ := result["success"].(bool)
    output, _ := result["output"].(string)
    return Result{success, output}
}
```

## Phase C — Daemon Mode (Tasks C1-C3)

### Task C1: Kernel daemon process

**Files:** `crates/ck-kernel/src/main.rs`

The kernel listens on a control pipe for commands. `ck start` connects and sends a `CreateTask` command, then disconnects. The kernel keeps running.

- [ ] **Step 1: Add daemon control pipe to kernel**

In `crates/ck-kernel/src/main.rs`, modify to:
1. Accept `--daemon` flag via `std::env::args()`
2. In daemon mode: create a control pipe (`ck-control`), accept commands in a loop
3. Commands arrive as JSON: `{"cmd": "create_task", "goal": "..."}` or `{"cmd": "shutdown"}`
4. Forward to `cmd_tx` channel

```rust
use tokio::sync::mpsc;
use ck_kernel::config::KernelConfig;
use ck_kernel::runtime::{Runtime, RuntimeCommand};

#[tokio::main]
async fn main() {
    tracing_subscriber::fmt().json().init();

    let args: Vec<String> = std::env::args().collect();
    let daemon_mode = args.contains(&"--daemon".to_string());

    let config = KernelConfig::default();
    let cognition_pipe = config.cognition_pipe.clone();
    let worker_pipe = config.worker_pipe.clone();
    let (cmd_tx, cmd_rx) = mpsc::channel::<RuntimeCommand>(64);

    let mut runtime = Runtime::new(config.clone(), cmd_rx).expect("failed to init runtime");

    let tx = cmd_tx.clone();
    tokio::spawn(async move {
        let _ = tokio::signal::ctrl_c().await;
        let _ = tx.send(RuntimeCommand::Shutdown).await;
    });

    if daemon_mode {
        // Listen for CLI commands on control pipe
        let control_pipe_name = "ck-control";
        let control_server = ck_ipc::server::PipeServer::new(control_pipe_name);
        let tx = cmd_tx.clone();
        
        tokio::spawn(async move {
            loop {
                match control_server.listen() {
                    Ok(listener) => {
                        if let Ok(mut conn) = listener.accept().await {
                            loop {
                                let msg: Result<serde_json::Value, _> = conn.read().await;
                                match msg {
                                    Ok(v) => {
                                        match v["cmd"].as_str() {
                                            Some("create_task") => {
                                                if let Some(goal) = v["goal"].as_str() {
                                                    let _ = tx.send(RuntimeCommand::CreateTask {
                                                        goal: goal.to_string()
                                                    }).await;
                                                }
                                            }
                                            Some("shutdown") => {
                                                let _ = tx.send(RuntimeCommand::Shutdown).await;
                                                return;
                                            }
                                            _ => {}
                                        }
                                    }
                                    Err(_) => break, // client disconnected
                                }
                            }
                        }
                    }
                    Err(e) => {
                        tracing::warn!("control pipe error: {e}");
                        tokio::time::sleep(tokio::time::Duration::from_secs(1)).await;
                    }
                }
            }
        });
    }

    runtime.connect_workers(&cognition_pipe, &worker_pipe).await;
    runtime.run().await;
}
```

```rust
pub async fn cmd_daemon() {
    let config = KernelConfig::default();

    // Build Go worker if needed
    let worker_bin = std::path::Path::new(&config.worker_bin);
    if !worker_bin.exists() {
        println!("Building Go worker...");
        let _ = std::process::Command::new("go")
            .args(["build", "-o", &config.worker_bin, "./cmd/ck-worker"])
            .current_dir("workers")
            .status();
    }

    let pipe_prefix = r"\\.\pipe\";
    let cognition_pipe_path = format!("{}{}", pipe_prefix, config.cognition_pipe);
    let worker_pipe_path = format!("{}{}", pipe_prefix, config.worker_pipe);

    // Spawn kernel as background process
    println!("Starting Cognition Kernel daemon...");
    let kernel_bin = std::env::current_exe()
        .ok()
        .and_then(|p| p.parent().map(|d| d.join("ck-kernel.exe")))
        .unwrap_or_else(|| std::path::PathBuf::from("ck-kernel.exe"));

    // Spawn workers first (daemon mode: kernel waits for them)
    let mut supervisor = Supervisor::new("ck");
    supervisor.spawn_tool_worker(&config.worker_bin, &worker_pipe_path).ok();

    let cognition_dir = std::path::Path::new(&config.cognition_script)
        .parent().and_then(|p| p.parent())
        .map(|p| p.to_string_lossy().into_owned())
        .unwrap_or_else(|| "cognition".into());
    supervisor.spawn_cognition_module(&config.python_bin, &cognition_dir, &cognition_pipe_path).ok();

    // Launch kernel in daemon mode (detached process)
    std::process::Command::new(&kernel_bin)
        .arg("--daemon")
        .spawn()
        .expect("failed to start kernel daemon");

    println!("Daemon started. Use `ck start \"<goal>\"` to send tasks.");
    println!("Use `ck stop` to shut down the daemon.");
}
```
